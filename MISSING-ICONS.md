## Fehlend

### Prio 1 -- Storefront Essentials

| Icon | Zweck |
|---|---|
| **Cart / Shopping Bag** | Warenkorb in der Navbar, Badge-Bounce bei Add-to-Cart |
| **Trash / Delete** | Artikel aus Warenkorb entfernen |
| **Checkmark / Success** | Add-to-Cart Bestaetigung, Formular-Validierung |
| **Arrow / Chevron** | Zurueck-Navigation, Dropdown, Accordion, Carousel |

### Prio 2 -- Nice to have

| Icon | Zweck |
|---|---|
| **User / Account** | Account-Menu, Login |
| **Star** | Produkt-Bewertungen |
| **Share** | Produkt teilen |
| **Copy / Clipboard** | Gutscheincode kopieren |
| **Plus / Minus** | Quantity Stepper (falls nicht als eigene Komponente) |
| **Loading / Spinner** | Async-Actions, Daten laden |
| **Truck / Delivery** | Versandstatus-Anzeige |
| **Tag / Discount** | Sale-Badge, Rabatt-Aktionen |

### Nicht noetig (Medusa/Stripe)

| Icon | Grund |
|---|---|
| Credit Card / Payment | Stripe Payment Element rendert eigene UI |
| Lock / Secure | Stripe zeigt eigene Sicherheits-Badges |
| Package / Box | Medusa Order-Seiten haben eigene Darstellung |
